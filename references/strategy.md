# A杀抓启动 — 策略原文参考

## 策略来源

B站视频：BV1zxRuBZEH6《A杀抓启动》

## 核心逻辑

近期市场出现大量"一杀下来就横盘"的股票，横盘后一直磨。本文详解如何做这类票。

### 选股条件

1. **A杀下来后横盘** — 大幅下跌后在底部横住
2. **市值 ≤ 60亿** — 小市值弹性大
3. **不能在下跌过程中介入**
4. **横盘至少 ≥ 20个交易日**
5. 横盘期间有明显放量上行信号

### 为什么不能在放量阳线当天买入？

因为放量大阳线当天，绝大部分形态一定会再次回落（回调）。
原因：经过长时间横盘，里面不坚定的筹码已走完，剩下顽固的。
主力通过连续拉升把剩下顽固筹码也清理掉。
如果这样都不走，再搞一段下跌——这就是**洗盘三部曲**。

### 两种介入形态

**形态① 冲高回踩型**
- 放量长阳拉升后
- 回调至放量长阳的中下部位置
- 不破放量阳线底部 → 入场

**形态② 破位反转型**
- 跌破横盘底部（假破位）
- 但没事，破了就可以入场
- 或者等企稳后买入

### 操作技巧

- 找前两个交易日成交量是近20个交易日最大成交量的票
- 以放量长阳的底部作为防守位（止损位）
- 大部分票在即将启动阶段，稍有犹豫就过去了
